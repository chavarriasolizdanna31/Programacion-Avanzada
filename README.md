# Programación Avanzada - Portafolio de Prácticas

Este repositorio contiene las actividades y tareas realizadas en la materia de Programación Avanzada. Aquí estaré subiendo los ejercicios de Programación Orientada a Objetos (POO) desarrollados en C++.

## 🚀 Contenido de la Clase (12/03/2026)

En la sesión de hoy aprendimos sobre el **Encapsulamiento** y la gestión de repositorios con **GitHub**.

### Ejercicios Realizados:
1. **Clase Autobús:** Práctica inicial de atributos privados y métodos públicos (getters y setters).
2. **El Guardián de la Torre:** Implementación de una clase `Torre` que valida movimientos en un tablero de ajedrez (8x8).
3. **Piedra, Papel o Tijera (Versión POO):** Juego que utiliza clases para gestionar las jugadas y el puntaje de los jugadores.

## 🛠️ Herramientas Utilizadas
* **Lenguaje:** C++
* **IDE:** ZinjaI / Visual Studio Code
* **Control de Versiones:** Git y GitHub Desktop

## 📝 Notas de Aprendizaje
* Los atributos **privados** ayudan a proteger los datos.
* Usamos `this->` para referirnos a los atributos de la propia clase dentro de sus métodos.
* GitHub nos permite guardar un historial de cambios (Commits) y compartir nuestro código en la red (Push).
